Assignment 22: Turf and Proj JS Playground

After looking at Turf website, I've identified 5 tools/functions that could be useful in a future project:

  • lineDistance: lineDistance takes a LineString & measures its length in the specified units. This would be cool to use in a project where you allow the user to draw their own lines, where they could potentially draw a line and want to know the total distance of line, for something like a road trip.
  • distance: distance calcs the distance between points in degrees, radians, miles, or kilometers. This would also be helpful in a mapping project because it allows for more user interaction and would allow the user to find out the distance between two points, especially in a transit map, for example.
  • truncate: truncate takes a GeoJSON Feature or FeatureCollection and truncates the precision of the decimals to a number you can choose. This would be helpful in maps where you allow the users to click on the map and display their coordinates without displaying a very long decimal that is overwhelming to a user.
  • random: random generates random GeoJSON data of both points and polygons. This would be especially helpful in testing while you create and generate projects as it allows markers and polygons to be added quickly.
  • bbox: bbox takes a set of features and calculates the box around all input features, then returns a bounding box. This would be good to include in a project if you wanted to focus in on a specific area and call attention to all markers in one area. The user's eye would be instantly drawn to the box and could see the points more quickly.
